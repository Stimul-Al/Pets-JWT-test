### General description

* Using technology:
  * Spring-Boot 
  * Spring-Security-JWT 
  * PostgreSQL 
  * Liquibase 
  * Mapstruct
  * Spring-Scheduling
  * Honeycomb
* A person can have several animals
* Animal can have several owners
* Animals are dogs or cats
* Removing a person leads to the removal of his animals
* Owners can exchange animals
